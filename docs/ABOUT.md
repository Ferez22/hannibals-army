# General Information

## Vision

This project is an agentic system. The goal is to create a digital twin of myself, that can plan/act on one's behalf in various scenarios.

## Current State

The project is in its early stages. It is not yet ready for production.
The next milestone to reach is to have a basic digital twin working, able to answer simple questions about one's self.

### Use Case Examples

- Help plan a(n) trip/studies/event
- Help with daily tasks management (access to agenda and notes apps)
- Help with daily reminders (via messaging platforms)
